# The Global Launch 🌍

A static portfolio website demonstrating cloud hosting concepts by deploying without provisioning any servers.

## 📌 Scenario
As a freelance developer, I built a personal portfolio website for a client who needed global reach without expensive servers. The goal was instant loading worldwide using cloud storage.

## 🚀 Live Demo
[https://devgambo-code9.github.io/the-global-launch](https://devgambo-code9.github.io/the-global-launch)

## 🎯 Mission Completed
- [x] **Cloud Storage Hosting**: Deployed using GitHub Pages - equivalent to AWS S3 Static Hosting / Azure Blob Storage
- [x] **Zero Server Provisioning**: Static HTML/CSS only. No backend, no server costs
- [x] **Global Accessibility**: Served via GitHub CDN for fast load times worldwide
- [x] **Public Access**: Configured for public read access

## 🛠️ Tech Stack
- **Hosting**: GitHub Pages 
- **Frontend**: HTML5, CSS3, Responsive Design
- **Concepts**: Static Website Hosting, Cloud Storage, CDN
